Linux常用命令

Levi Tang 2020-09-25 16:21:05
DevOps > Linux

查看进程

ps -ef 和 ps aux

格式有所不同

# 标准输出
# ps -ef
UID     => 用户ID、但输出的是用户名 
PID     => 进程的ID 
PPID    => 父进程ID 
C       => 进程占用CPU的百分比 
STIME   => 进程启动到现在的时间 
TTY     => 该进程在那个终端上运行，若与终端无关，则显示? 若为pts/0等，则表示由网络连接主机进程。 
CMD     => 命令的名称和参数

# BSD语法
# ps aux
USER       => 用户名 
%CPU       => 进程占用的CPU百分比 
%MEM       => 占用内存的百分比 
VSZ        => 该进程使用的虚拟內存量（KB） 
RSS        => 该进程占用的固定內存量（KB）（驻留中页的数量） 
STAT       => 进程的状态 
START      => 该进程被触发启动时间 
TIME       => 该进程实际使用CPU运行的时间

树形格式查看

pstree

ps -ejH

ps axjf

查看内存使用前十进程

ps aux|head -1;ps aux|grep -v PID|sort -rn -k +4|head

查看cpu使用前10的进程

ps aux|head -1;ps aux|grep -v PID|sort -rn -k +3|head
