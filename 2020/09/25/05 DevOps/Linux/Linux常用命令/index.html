查看进程

ps -ef 和 ps aux

格式有所不同

# 标准输出
# ps -ef
UID     => 用户ID、但输出的是用户名 
PID     => 进程的ID 
PPID    => 父进程ID 
C       => 进程占用CPU的百分比 
STIME   => 进程启动到现在的时间 
TTY     => 该进程在那个终端上运行，若与终端无关，则显示? 若为pts/0等，则表示由网络连接主机进程。 
CMD     => 命令的名称和参数

# BSD语法
# ps aux
USER       => 用户名 
%CPU       => 进程占用的CPU百分比 
%MEM       => 占用内存的百分比 
VSZ        => 该进程使用的虚拟內存量（KB） 
RSS        => 该进程占用的固定內存量（KB）（驻留中页的数量） 
STAT       => 进程的状态 
START      => 该进程被触发启动时间 
TIME       => 该进程实际使用CPU运行的时间

查看内存使用前十进程

ps aux|head -1;ps aux|grep -v PID|sort -rn -k +4|head

查看cpu使用前10的进程

ps aux|head -1;ps aux|grep -v PID|sort -rn -k +3|head

-t 查看tcp

-p 查看pid

vim操作相关

移动和查看

gg 回到回到第一列

G 跳到最后一行

20 + Enter 跳到第二十行

Ctrl + f 相当与pagedown

Ctrl + b 相当于pageup

?word 向上查找word字符串

编辑

yy 复制当前所在列

2yy 向下复制两行

u 撤销上个操作

Ctrl + r 反撤销上个操作

o 在当前下一列输入

进程相关

Ctrl + z 挂起相关进程

jobs 查看挂起的进程

fg %n 继续打开第n个进程

➜  ~ vim ~/.zshrc                                                                                                   
[1]  + 30917 suspended  vim ~/.zshrc

➜  ~ jobs   
[1]  + suspended  vim ~/.zshrc

➜  ~ fg %1

光标移动

ctr l+ 左右键 : 在单词之间跳转

ctrl + a : 跳到本行的行首

ctrl + e : 跳到页尾

Ctrl + u ：删除当前光标前面的文字 （还有剪切功能）

ctrl + k ：删除当前光标后面的文字(还有剪切功能)

Ctrl + L：进行清屏操作

Ctrl + y : 粘贴Ctrl+u或ctrl+k剪切的内容
