manjaro配置

Levi Tang 2020-12-31 11:17:15

安装应用

修改自定义目录

vim ~/.config/user-dirs.dirs

# ~/.config/user-dirs.dirs
XDG_DESKTOP_DIR="$HOME/desktop"
XDG_DOWNLOAD_DIR="$HOME/downloads"
XDG_TEMPLATES_DIR="$HOME/applications"
XDG_PUBLICSHARE_DIR="$HOME/public"
XDG_DOCUMENTS_DIR="$HOME/documents"
XDG_MUSIC_DIR="$HOME/music"
XDG_PICTURES_DIR="$HOME/pictures"
XDG_VIDEOS_DIR="$HOME/videos"

QQ

yay -S com.qq.im.deepin

Wechat

yay -S com.qq.weixin.spark

没有截图和透明边框问题，但是输入中文显示成了小方块，要安装微软雅黑字体

安装字体

把字体复制到/usr/share/fonts, 然后运行fc-cache --force刷新缓存
， yay -S freetype2-ultimate5安装一个字体的插件，可以更清晰

安装Postman

官网直接下载压缩包，放到～/Applications中，运行sudo ln -s ~/applications/Postman/Postman /usr/bin/postman 建立软链接，就可以直接命令postman启动

创建桌面图标

# 全局配置
vim /usr/share/applications/Postman.desktop

# 个人配置
vim ~/.local/share/applications/Postman.desktop

# Postman.desktop

[Desktop Entry]
Type=Application
Comment=postman
Name=Postman
Icon=/home/levi/applications/Postman/app/resources/app/assets/icon.png
Exec=/home/levi/applications/Postman/Postman
Terminal=false
Categories=Application;

有了桌面图标就可以添加到latte-dock

安装AppImageLauncher

设置AppImage的目录为~/applications就能读取到应用，然后再添加到dock

设置vim

在/etc/vimrc设置使用系统剪切板

set clipboard=unnamed,unnamedplus

unamed为可用选区的剪切板
unamedplus为可用普通的剪切板（只有linux需要设置）

安装auto-suggestion导致复制很慢

在~/.zshrc复制这段代码

# This speeds up pasting w/ autosuggest
# https://github.com/zsh-users/zsh-autosuggestions/issues/238
pasteinit() {
  OLD_SELF_INSERT=${${(s.:.)widgets[self-insert]}[2,3]}
  zle -N self-insert url-quote-magic # I wonder if you'd need `.url-quote-magic`?
}

pastefinish() {
  zle -N self-insert $OLD_SELF_INSERT
}
zstyle :bracketed-paste-magic paste-init pasteinit
zstyle :bracketed-paste-magic paste-finish pastefinish
