sql常用命令

Levi Tang 2020-12-20 13:15:24

常用sql命令

postgres

设置id从表最大id开始自动增长

select setval('tablename_id_seq',(select max(id) from tablename))

数据备份与回复

数据备份

pg_dump -v -U postgres -h 127.0.0.1 -p 5432 database > database.sql

-v 详细输出

-U 用户

-h 连接地址

-p端口号

database 数据库名

恢复数据

psql -h localhost -U postgres -d databasename <  database.sql

mysql

数据备份和恢复

备份

docker exec -it af10fce234a3 mysqldump -uroot -h192.168.0.50 -p123456 --databases tablename1 tablename2 > tablename.sql

备份出来的sql文件第一行会有提示语句，直接导入会报语法错误，要先删掉提示语
-all-databases 备份所有数据库

恢复

docker exec -i af10fce234a3 mysql -h127.0.0.1 -uroot -p123456 < skyland_jlfdl_admin.sql
