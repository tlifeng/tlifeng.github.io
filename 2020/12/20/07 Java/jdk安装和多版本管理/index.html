jdk安装和多版本管理

Levi Tang 2020-12-20 13:21:10

jdk安装和多版本管理

arch或manjaro

注意Manjaro系统只能安装openjdk

安装oraclejdk使用时报错，Unrecognized option: –version Error: Could not create the Java Virtual Mach

wiki文档地址：https://wiki.archlinux.org/index.php/Java

pacman安装

sudo pacman -S jdk8-openjdk

安装后，jdk所在目录/usr/lib/jvm/java-8-openjdk

manjaro/archlinux本身有一个管理jdk的命令

archlinux-java <COMMAND>

COMMAND:
    status        List installed Java environments and enabled one
    get            Return the short name of the Java environment set as default
    set         <JAVA_ENV>    Force <JAVA_ENV> as default
    unset        Unset current default Java environment
    fix            Fix an invalid/broken default Java environment configuration

其他linux系统

下载openjdk地址 http://openjdk.java.net/install/

下载oraclejdk地址 https://www.oracle.com/java/technologies/javase-downloads.html

下载后解压缩

tar -xzv -f ./jdk-8u261-linux-x64.tar.gz -C /usr/lib/jvm/jdk_8

-x 解压文件

-z 解压gzip压缩的文件，即.tar.gz结尾

-v 显示解压过程

-f 要处理的压缩包名称

-C 要解压的目录

安装jenv版本工具

# 下载jenv源文件
git clone https://github.com/jenv/jenv.git ~/.jenv

# 安装
# bash
echo 'export PATH="$HOME/.jenv/bin:$PATH"' >> ~/.bash_profile
echo 'eval "$(jenv init -)"' >> ~/.bash_profile
source ~/.zshrc

# zsh
echo 'export PATH="$HOME/.jenv/bin:$PATH"' >> ~/.zshrc
echo 'eval "$(jenv init -)"' >> ~/.zshrc
source ~/.zshrc

# jenv 使用
# jenv添加刚才安装的jdk,后面为jdk安装的目录
jenv add /usr/lib/jvm/jdk_8
# jenv查看所有管理的版本
jenv versions
# jenv查看当前使用版本
jenv version
# jenv全局设置某个版本
jenv global 1.8
# jev当前目录设置某个版本
jenv local 1.8
# jenv 设置shell使用的版本
jenv shell 1.8
# 查看jenv的更多命令
jenv commands
# 查看命令帮助
jenv help <command>
