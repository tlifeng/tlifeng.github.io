Vim常用操作

Levi Tang 2020-12-20 13:01:05

vim操作相关

移动和查看

h 左

l 右

k 上

j 下

w/W 跳过一个单词

b/B 返回一个单词

} 跳过一个段落

{ 返回一个段落

]] 相当于G

[[ 相当于gg

gk 在一个很长的段落中移动这个光标到上面

gj 在一个很长的段落中移动这个光标到下面

0 返回这行的句首

$ 返回这行的句尾

gg 回到回到第一列

G 跳到最后一行

20 + Enter 跳到第二十行

Ctrl + f 相当与pagedown

Ctrl + b 相当于pageup

?word 向上查找word字符串

编辑

yy 复制当前所在列

2yy 向下复制两行

u 撤销上个操作

Ctrl + r 反撤销上个操作

o 在当前下一列输入

复制剪切板的内容

配置vimrc, 一般全局的配置在/etc/vimrc文件，个人配置在~/.vimrc。

set clipboard=unnamed,unnamedplus

参考链接

编辑时使用root权限保存

在命令模式下，输入

:w !sudo tee %

进程相关

Ctrl + z 挂起相关进程

jobs 查看挂起的进程

fg %n 继续打开第n个进程

➜  ~ vim ~/.zshrc                                                                                                   
[1]  + 30917 suspended  vim ~/.zshrc

➜  ~ jobs   
[1]  + suspended  vim ~/.zshrc

➜  ~ fg %1
