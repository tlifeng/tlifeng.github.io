Git常用命令

Levi Tang 2020-12-20 13:18:19

Git使用

git下载慢

加一个cnpmjs.org的后缀

 git clone https://github.com.cnpmjs.org/grpc/grpc.git

设置代理

git config --global https.proxy http://127.0.0.1:8889

记忆凭证

git config --global credential.helper store

remote — 远程仓库管理

  • 显示远程仓库的URL
git remote -v
  • 添加远程, 创建一个标记为upstream，仓库为git://url的仓库源
git remote add upstream git://url
  • 显示upstrem仓库状态
git remote show upstream
  • 拉取upstrem仓库对应master分支（当前项目会生成upstream/master分支）
git fetch upstream master
  • 合并分支代码
git merge upstream/master
  • 推送到远程仓库
git push origin master
  • 删除远程仓库
git remote remove upstream
  • 修改远程仓库名upstream为up
git remote rename upstream up

cherry-pick 合并需要的commit

当前分支merge需要的commit

git cherry-pick <commitHash>

merge从a到b的commit

git cherry-pick a..b

stash - 临时保存

git stash //临时保存
git stash pop [num] //提取记录。num为可选项，第几次记录
